PsiGestao Backend

API de produção em /api para usuários, livro-caixa, importação/exportação e dashboard.

API ativa

Autenticação/Usuários

  • GET /api/users
  • POST /api/users
  • PUT/PATCH/DELETE /api/users/:id

Livro Caixa

  • GET /api/transactions
  • POST /api/transactions
  • PUT/PATCH/DELETE /api/transactions/:id

Planilhas

  • POST /api/transactions/import
  • GET /api/transactions/export?format=csv|xlsx|json

Dashboard e Relatórios

  • GET /api/dashboard/summary
  • GET /api/dashboard/charts
  • GET /api/reports/dre
